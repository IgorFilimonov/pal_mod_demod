# PAL_MOD_DEMOD

**PAL_MOD_DEMOD** — утилита, совмещающая в себе генератор и приемник PAL сигнала.

## Генератор

Источником для генерации PAL сигнала является цифровое видео. При этом поддержаны все форматы, указанные по [ссылке](https://ffmpeg.org/ffmpeg-formats.html).

Для запуска генератора воспользуйтесь командой 

```python pal_mod.py <input_filename> <output_filename> [sample_rate [frames_amount]]```

где `input_filename`, `output_filename` --- файлы с входным видео и выходным PAL сигналом соответственно, а `sample_rate` и `frames_amount` --- частота дискретизации и количество кадров, которые будут модулироваться (опционально, значения по умолчанию 16000000 и все видео).

## Приемник

Для запуска приемника воспользуйтесь командой

```python pal_demod.py <input_filename> [sample_rate] [hsync vsync [level_sync [invert_video]]]```

где `input_filename` --- записанный PAL сигнал, а `sample_rate`, `hsync` и `vsync`, `level_sync`, `invert_video` --- частота дискретизации, использовать или нет горизонтальную и вертикальную синхронизации, уровень синхронизации, инвертировать изображение или нет (опционально, значения по умолчанию 16000000, нет, 0.45, нет).

## Тестирование

Для тестирования генератора используются 10 видеофайлов формата `mp4`, расположенные в `tests/resources/mod`. Для запуска тестов генератора воспользуйтесь командой

```python test_mod.py```

Для тестирования приемника используются 10 сгенерированных PAL сигналов, а также 3 записанных отдельно. Для запуска тестов приемника воспользуйтесь командой

```python test_demod.py```
